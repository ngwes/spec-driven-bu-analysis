# Conversational Tasks

This is a DIALOG. Break the spec into tasks one user story at a time.

1. Read `spec.md` for the current feature.
2. Break down the first user story into tasks.
3. Present the tasks to the user:
   "Here's how I'd break down User Story 1. Does this look right?"
4. If the user wants changes, adjust. Otherwise move to the next story.
5. After all stories are broken down, present the complete tasks.md.
6. Ask: "Anything to reorder or add before this is final?"
