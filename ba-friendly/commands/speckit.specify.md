# Conversational Specify

This is a DIALOG, not a one-shot. Ask ONE question at a time.

1. Ask the user what feature they want to build.
2. Ask ONE follow-up question per turn. Work through:
   - Who is this for? (user, persona, role)
   - What should happen, step by step? (user journey)
   - Are there business rules or constraints?
   - How will you know it's done? (acceptance criteria)
3. When the user indicates readiness, produce the final spec.md.
